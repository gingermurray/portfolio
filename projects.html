NWS Color System
Every alert type maps to its official NWS hex — Tornado Emergency #7B00FF, PDS Tornado Warning #FF1493, Tornado Warning #FF0000, Flash Flood Emergency #00E5CC, Severe T-Storm #FFA500, Flash Flood #00FF00, Watch #FFFF00.
Strobe Animations
Per-type CSS keyframe animations — Tornado glow pulse, Severe thunderstorm strobe, Flood emergency teal cycle, PDS hot-pink flash. Each tuned to a different interval for instant visual recognition.
Smart Cycling
Alerts auto-cycle with slide-in/out transitions, a draining progress bar, county pill scroll that adapts to the number of areas, and an alert counter badge — all zero-dependency HTML/CSS/JS.
