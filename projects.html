Project 03 · Featured Build

NWS Alert Card
Interactive Overlay

An OBS-inspired interactive weather alert system built entirely in pure HTML, CSS, and JavaScript — zero dependencies. Pulls live data from api.weather.gov, maps every alert type to its official NWS hex color, and fires per-type strobe animations for instant visual recognition. Cycles through multiple active alerts with smooth transitions and a draining progress bar.

NWS Color System
Every alert type maps to its official NWS hex — Tornado Warning #FF0000, Severe T-Storm #FFA500, Watch #FFFF00, PDS #FF1493.
Strobe Animations
Per-type CSS keyframe animations — Tornado glow pulse, Severe thunderstorm strobe, Flood emergency teal cycle, PDS hot-pink flash. Each tuned to a different interval for instant visual recognition.
Smart Cycling
Alerts auto-cycle with slide-in/out transitions, a draining progress bar, county pill scroll that adapts to the number of areas, and an alert counter badge — all zero-dependency HTML/CSS/JS.
